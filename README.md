# FursealClient
The Furseal client using Electron.
This just a simple UI for Furseal core. we need your help to make it cooler.

## Dependency

* node.js
* npm

## Build & Install

```shell
cd /path/to/FursealClient
git clone https://github.com/TheFurseal/Furseal.git
npm i
npm start
```
