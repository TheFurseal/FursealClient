# FursealClient
The Furseal client uses Electron.
This is just a simple UI for Furseal core. We would love to have your help to make it cooler.

## Dependency

* node.js
* npm

## Build & Install

```shell
cd /path/to/FursealClient
git clone https://github.com/TheFurseal/Furseal.git
npm i
npm start
```
